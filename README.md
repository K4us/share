# K4us Share

## Khmer Chess Font Family

`./font`

## Khmer Chess Font SVG helper

A Typescript and Node.js base helpers for getting Khmer Chess SVG.

usage:

```bash
$ npm i k4us-share
```

Typescript

```typescript
import {
   boatSVG,
   fishSVG,
   generalSVG,
   horseSVG,
   kingSVG,
   queenSVG,
   transformFishSVG,
} from 'k4us-share/types';
```

Node.js

```javascript
const { khmerChessPieces } = require('k4us-share');
const {
    kingSVG,
    queenSVG,
    generalSVG,
    horseSVG,
    boatSVG,
    fishSVG,
    transformFishSVG,
} = khmerChessPieces;
```
